# 🎨 Ejemplo Completo: Flexbox Layout Profesional

## 📋 Descripción
Ejemplo práctico de un layout completo usando Flexbox con navbar responsive y diseño moderno.

## 🎯 Conceptos Demostrados

### Navbar con Flexbox
```css
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
```

### Hero Section
```css
.hero {
    display: flex;
    align-items: center;
    gap: 3rem;
}
```

### Grid de Tarjetas
```css
.features {
    display: flex;
    gap: 2rem;
}
```

### Layout Sidebar + Contenido
```css
.content-section {
    display: flex;
    gap: 2rem;
}
```

## 🚀 Cómo Usar

1. Abre `index.html` en tu navegador
2. Redimensiona la ventana para ver el comportamiento responsive
3. Inspecciona el código CSS para entender cada propiedad Flexbox

## 📱 Características

- ✅ Navbar sticky con gradiente
- ✅ Hero section con call-to-action
- ✅ Tarjetas de características con hover effects
- ✅ Layout sidebar + contenido principal
- ✅ Footer multi-columna
- ✅ 100% responsive
- ✅ Animaciones suaves

## 🎓 Propiedades Flexbox Utilizadas

- `display: flex`
- `justify-content: space-between`
- `align-items: center`
- `flex-direction: column`
- `gap: 2rem`
- `flex: 1`
- `flex-wrap: wrap`

## 💡 Aprende Más

Este ejemplo demuestra cómo Flexbox simplifica la creación de layouts complejos sin necesidad de floats o posicionamiento absoluto.
